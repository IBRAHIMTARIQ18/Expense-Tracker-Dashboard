# SpendWise — Personal Finance Dashboard

A modern and responsive expense tracking dashboard built with React, TypeScript, Bootstrap, and Zod. SpendWise helps users manage expenses, organize spending categories, visualize financial data, and gain insights through analytics.

---

## Live Demo

**Live Website:** (https://spendwise-dashboard.netlify.app/)

---

## Features

- Add, delete, and manage expenses dynamically
- Filter expenses by category
- Real-time KPI summary cards
- Expense analytics visualization using charts
- Form validation using Zod
- Responsive dashboard UI for desktop, tablet, and mobile
- Animated UI with Framer Motion
- Glassmorphism-inspired modern design
- Dynamic calculations for:
  - Total Expenses
  - Number of Transactions
  - Largest Expense
  - Total Categories

- Interactive expense table with filtering
- Smooth animated background gradient

---

## Tech Stack

- ReactJS
- TypeScript
- Bootstrap
- Vite
- Zod Validation
- Framer Motion
- Recharts (Charts & Analytics)
- CSS3

## Installation & Setup

Clone the repository:

```bash
git clone <your-repo-link>
```

Move into the project folder:

```bash
cd spendwise-dashboard
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

The project will run locally on:

```bash
http://localhost:5173
```
